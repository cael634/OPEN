# 📦 OPEN – Plataforma de Logística y Pedidos para Empresas de Alimentos

**OPEN** es una plataforma web y móvil que digitaliza el proceso de ventas, pedidos, entregas y estadísticas para empresas del sector alimenticio en Latinoamérica. Más de 30.000 descargas respaldan su impacto en pequeñas, medianas y grandes empresas.

---

## 🚀 ¿Qué es OPEN?

OPEN conecta empresas distribuidoras de alimentos con mayoristas, tiendas, minimercados y supermercados, ofreciendo herramientas digitales para:

- Tomar y administrar pedidos
- Coordinar entregas en rutas optimizadas
- Visualizar estadísticas de ventas
- Mejorar la comunicación con clientes
- Controlar cuentas, rutas y productos desde un panel web

---

## 📱 Características principales

### 👥 Para clientes
- Catálogo disponible las 24 horas desde cualquier dispositivo
- Pedidos con programación y estado en tiempo real
- Historial de compras y estadísticas personales
- Comunicación directa con distribuidores

### 🧑‍💼 Para empresas
- Panel web de administración (productos, usuarios, pedidos, rutas)
- Estadísticas detalladas por producto, día, semana y mes
- Gestión de cuentas personalizadas de clientes
- Asignación automática de rutas y canales de venta

### 🚚 Para repartidores
- App móvil con rutas asignadas y pedidos por entregar
- Información detallada del pedido y del cliente
- Monitoreo en tiempo real del estado de las entregas

---

## 📊 Módulos disponibles

- 🛍️ Pedidos y catálogo online
- 📈 Estadísticas por producto y canal
- 📦 Administración de rutas y entregas
- 📲 Aplicaciones móviles para ventas y distribución
- 🔄 Conectividad entre clientes, vendedores y distribuidores
- 💬 Chat y notificaciones para clientes

---

## 🧰 Tecnologías utilizadas

- **Frontend móvil:** Flutter
- **Backend:** Firebase (Firestore, Authentication, Cloud Messaging)
- **Panel web:** HTML, CSS, JavaScript
- **Mapas y rutas:** Google Maps API
- **Otros:** Firebase Realtime Database, estadísticas integradas

---

## 📍 Visión

Facilitar la transformación digital de empresas de alimentos en Latinoamérica, cubriendo necesidades logísticas, comerciales y estadísticas mediante una plataforma intuitiva y escalable.
